{% block title %}Overview (part 1/5){% endblock %}

In this study, we would like to collect behavioral data of teamwork in gamified tasks. The study session is divided into five parts

  1. An overview of the study. (1 min)
  2. A short demographic survey. (1 min)
  3. Team Task A. (10-13 min)
  4. Team Task B. (7-9 min)
  5. A short post-experiment questionnaire. (1 min)

For each task, you will be first given a TUTORIAL to describe the task. Then, you will complete FOUR actual TASK sessions.
