{% block title %}Overview (part 1/5){% endblock %}

In this study, we would like to collect behavioral data according to the destination through gamified tasks. The study session is divided into five parts

  1. An overview of the study. (1 min)
  2. A short demographic survey. (1 min)
  3. Task A: Working with a robot to move boxes in a room. (7-9 min)
  4. Task B: Working with a fire truck to rescue people. (7-9 min)
  5. A short post-experiment questionnaire. (1-2 min)

For each of the two tasks, you will be first given a TUTORIAL to describe the task. Then, you will complete FOUR actual TASK sessions.

The tasks are designed to contain multiple targets to deal with. To complete the tasks as efficiently as possible, you may need to change your target during the task. Whenever the target is changed in your mind, you are supposed to consciously mark your destination via the provided user interface. You will learn how to use the user interface in the tutorial.
